==============================
How we generated our KPI's --->
==============================

----------------------------------------------------------------------------------------
Unique ID    

16 digit Unique Identifier Generated using 'UUID'


----------------------------------------------------------------------------------------
Region and Loca Authority 

'Region' is distributed according to the probability of regions in Source dataset
'Local Authority' is randomly chosen according to the generated 'Region' 


----------------------------------------------------------------------------------------
Geographic Code    

It is depending on the generated Local Authority


----------------------------------------------------------------------------------------
Sic Group    

Sic group is generated by the count and probability in Source dataset obtained from ONS


----------------------------------------------------------------------------------------
Sector    

It is generated according to the SIC group


----------------------------------------------------------------------------------------
Structure Type    

Using the count of Structure per Local Authority we derived probability of each structure
generated according to it.


----------------------------------------------------------------------------------------
Electricity and Gas    

Per Structure consumption = Total consumption by Local Authority / Structure count in Local
'Per structure consumption' is normally distributed accoding to structure type


----------------------------------------------------------------------------------------
Petrol / Natural Gas / CoalConsumption Calculations     

Total consumption for petrol = forecasted value using data from 1990 - 2018 (by sextor)
county percentage = 
Per county consumption	=   Total consumption for petrol * County percentage
overall structure consumption =  per county consumption * strcuture percentage
per_structure consumption = overall structure consumption / structure count

'Per structure consumption' is normally distributed accoding to structure type
